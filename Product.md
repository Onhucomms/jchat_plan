# Onhucomms의 새로운 채팅 서비스
> 새로운 채팅 서비스는 서비스 빌더로서 AI Agent를 최대한 이용하여 채팅 앱 서비스를 구현하는데 목적이 있음
> 되도록 AI를 이용하며 UI 및 각종 기능은 AI Agent로 생성한다.
> 채팅 서비스의 특징은 남녀의 성비를 맞추어 서비스 사용을 가능하게 함으로써 사용자의 이탈을 줄이고 신뢰높은 서비스 구현

## 기반기술
1. Go Lang을 이용한 API 서비스 제작
2. gRPC를 이용한 실시간 데이터 전송 서비스 구현
3. Nextjs를 이용한 어드민 구현
4. Flutter를 이용한 Android, IOS 앱 구현
5. Mysql를 이용한 각종 정보 저장
6. Docker를 이용한 서비스 배포
7. github을 이용한 버전관리

## 선행작업
1. Mysql을 이용한 DB 설계
2. Docker를 이용한 개발 환경 구성

## Product Summary
1. 회원
   - 회원은 회원가입 후 남/여 성비에 따라서 가입승인이 이루어짐
   - 최초 남자 50명, 여자 50명 까지는 가입승인 없이 바로 가입이 가능함.
   - 남/여 각 50명(전체 100명)이 모두 가입되게 되면 남/여 성비에 따라서 가입승인이 이루어짐. 
   - 가입승인 완료된 남자가 101명 이고 여자가 101명일 경우 신규가입 여자 1명이 등록될경우 여자는 가입승인이 자동으로 이루어지고 승인대기중인 남자중에 등록일이 오래된 순으로 가입승인 처리를 한다.
   - 회원은 가입시 본인인증을 통하여 성별,나이를 추출하고 20세 이상만 회원가입이 가능하다.
   - 회원가입시 메일주소, 비밀번호, 닉네임, 프로필(채팅리스트에 노출되는 내용)를 입력하는데 입력완료 후 "인증메일 보내기" 버튼을 클릭하면 인증메일을 발송하고 메일안의 링크를 클릭하면 회원가입이 완료된다.
   - 회원상태는 online, offline, 사용불가, 강제탈퇴, 본인탈퇴 등이 있음.
   - online 회원이 앱 구동이 3일이 지나면 자동으로 offline 처리되며 offline 회원은 채팅 가능 리스트에서 배제된다.
   - offline 회원이 앱을 구동하면 자동으로 online 처리됨
   - 회원과의 채팅시 신고를 누적 3회 받으면 사용불가 처리되고 사용불가 3회 받으면 자동으로 강제탈퇴 처리된다.
   - 회원가입완료시 승인대기 인원수를 보여주고 포인트를 이용한 "바로승인" 버튼을 보여주고 포인트 충전을 유도한다.
   - 바로승인 버튼을 클릭하면 200포인트를 차감하고 승인처리를 완료한다.
   - 비승인, 승인회원 모두 포인트를 누적할 수 있다.
   - 승인완료시 서비스를 이용할 수 있도록 500포인트를 부여한다.
   - 프로필 설정에서 프로필을 변경 할 수 있다.
2. 앱 실행 시 프로세스
   - 앱 로딩 시 현재의 위치정보를 추적하여 사용자의 현재위치를 지정한다.
   - 사용자의 현재위치는 5분에 한번씩 조회하여 사용자 위치를 업데이트 한다.
3. 채팅
   - 채팅은 1:1 채팅을 원칙으로 한다.
   - 채팅 리스트는 현재 online 회원만 노출한다.
   - 채팅 리스트의 순서는 기본으로 가장 가까운 거리 순으로 노출하고 상단의 소팅 아이콘을 클릭하면 가까운순, 채팅 횟수순, 좋아요 순으로 소팅순서를 변경할 수 있다.
   - 채팅 리스트에는 회원 썸네일(없을경우 기본 썸네일 노출), 닉네임, 프로필, 채팅횟수, 좋아요를 노출한다.
   - 채팅 리스트를 터치하면 채팅창이 열린다.
4. 포인트


2. 회원가입
   1) 회원
   2) 본인인증 : 서비스는 성인만 가입하여 사용가능하므로 본인인증 후 성인여부, 성별, 나이 정보를 추출 후 Decript 가능한 암호화를 통하여 암호화 하여 DB에 저장한다.
   3) 기본정보 입력
      - 메일주소 : 로그인 아이디 값으로 메일 주소를 이용하며 메일 발송 후 인증이 되어야만 회원가입이 완료된다.
      - 비밀번호 : 비밀번호는 영문(대소문자 포함) + 숫자 + 특수문자로 이루어진 6글자 이상 비밀 번호입력
      - 닉네임 : 닉네임은 영문,숫자,한글만 등록가능함.
      - 성별 : 성별은 본인인증 정보에서 가져와서 표시만 해준다.
   4) 메일주소 인증 : 메일로 보내온 인증키를 포함한 인증 URL을 클릭하면 자동으로 인증되고 인증이 완료되었을 경우
